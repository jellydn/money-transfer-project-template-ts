> Temporal Money Transfer example in TypeScript

## Pre-requisites

- [Bun](https://bun.sh)
- [Temporal CLI](https://docs.temporal.io/server/quick-install/)

## Install

```sh
bun install
```

## Usage

```sh
bun run dev
```
